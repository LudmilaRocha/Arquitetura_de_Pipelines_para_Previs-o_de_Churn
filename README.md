# Arquitetura_de_Pipelines_para_Previs-o_de_Churn
Previsão de Churn Acadêmico: ML além do Notebook 🎓  Solução de Data Science para retenção de alunos usando Pipelines do Scikit-Learn. O projeto encapsula todo o pré-processamento e o modelo (Random Forest) em um fluxo único, garantindo robustez e produção.  ✅ Zero Data Leakage ✅ CatBoostEncoding &amp; Scaling ✅ Foco em Negócio
